Snake
=======

Игра на Arduino и на светодиодной матрице 8х8.

+ Авторы: Vaicel, dubkov
+ Дата создания: 14.03.15
+ Лицензия: LGPL v2.1

-------

Это стандартная игра змейка, которая в процессе создания.Пока она не будет отличаться от оригинала.
